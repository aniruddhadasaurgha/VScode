Box Model Example

The Box Model is a core concept in CSS (Cascading Style Sheets) used to describe how elements are structured and displayed on a web page. Each HTML element is considered a rectangular box, and the Box Model breaks it down into four parts from the inside out:

  • 1.Content
  • This is where the text or images of the element appear.
  • Width and height properties directly affect this area.
  • 2.Padding
  • This is the space between the content and the border.
  • Padding can be set using the padding property.
  • 3.Border
  • This is a line that wraps around the padding (if any) and content.
  • Border properties can be set using the border property.
  • 4.Margin
  • This is the space outside the border, separating the element from others.
  • Margins can be set using the margin property.
