Pseudo Classes

CSS pseudo-classes are used to define the special state of an element.

For example, a link can be styled differently when it is hovered over or visited.

In this example, we will demonstrate the use of pseudo-classes with a simple link.

Visit Example

Hover over the link to see the effect of the :hover pseudo-class.

Click the link to see the effect of the :active pseudo-class.

After visiting, the link will change color due to the :visited pseudo-class.

When the link is focused, it will have a different style due to the :focus pseudo-class.

When the link is disabled, it will have a different style due to the :disabled pseudo-class.

When the link is enabled, it will have a different style due to the :enabled pseudo-class.

When the link is checked, it will have a different style due to the :checked pseudo-class.

When the link is not checked, it will have a different style due to the :not(:checked) pseudo-class.

When the link is first-child, it will have a different style due to the :first-child pseudo-class.

When the link is last-child, it will have a different style due to the :last-child pseudo-class.

When the link is nth-child, it will have a different style due to the :nth-child(2) pseudo-class.

When the link is nth-last-child, it will have a different style due to the :nth-last-child(2) pseudo-class.
