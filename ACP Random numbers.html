Random Numbers

This page generates 10 random numbers between 1 and 100 using a while loop.

Each number is generated using the formula: \[ \text{result} = \text{Math.floor(Math.random() * 100) + 1} \]

result = Math.floor(Math.random() * 100) + 1;

Where:

Math.random() generates a random number between 0 (inclusive) and 1 (exclusive).

Math.floor() rounds down to the nearest whole number.

The result is then adjusted to fit within the desired range of 1 to 100.
